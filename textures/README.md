# Textures

Drop optional ambience or noise files in this folder if you want to use:

```bash
./gfi --texture random input.mp3
```

Good candidates:

- vinyl crackle
- tape hiss
- rain beds
- room tone
- modem noise
- distant crowd ambience

Supported formats include `wav`, `mp3`, `flac`, `ogg`, `m4a`, and `aiff`.
